# OS_CourseWork

Сървър-клиент приложение, чрез което се играе модификация на класическата игра на бесеница, където двамата играчи избират дума, която другият трябва да отгатне. Този, който има по-малко грешки след като и двамата са отгатнали думата на другия, печели, а ако са с еднакъв брой грешки играта е равна.
